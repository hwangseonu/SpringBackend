# SpringBackend  
Spring boot 를 활용한 CRUD가 있는 백엔드를 구현해본다.   

## Technical Stack

### Language
- java 8

### Framework
- Spring Boot 2.1.3.RELEASE

### Dependency Management
- Gradle

### Database
- MariaDB

### ORM
- JPA (hibernate)

### Authentication
- Json Web Token (jjwt:0.9.0)

### Server
- Tomcat (embedded)
